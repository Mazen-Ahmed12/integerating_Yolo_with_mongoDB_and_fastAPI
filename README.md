# YOLO + MongoDB + FastAPI Integration

Real-time object detection API using YOLO and FastAPI, with MongoDB for data storage.

## Features
- FastAPI endpoints for detection
- YOLO inference engine
- MongoDB integration
- Lightweight and fast

## Tech Stack
- YOLO (Ultralytics)
- FastAPI
- MongoDB
- Python

## Use Case
Perfect for real-time detection services and backend CV pipelines.

## Installation & Launch

1. Clone the repo:
   ```bash
   git clone https://github.com/Mazen-Ahmed12/integerating_Yolo_with_mongoDB_and_fastAPI.git
   cd integerating_Yolo_with_mongoDB_and_fastAPI

2. Install dependencies:
   ```bash
   pip install -r requirements.txt

3. Launch FastAPI:
   ```bash
   uvicorn app:app --reload

4. Access API docs:
   - http://127.0.0.1:8000/docs
